# Projeto: Aplicação Desktop com JavaFX e JDBC: Curso Java completo 2023

## Itens abordados:

* Teoria sobre JavaFX

* Arquivos FXML

* Alert, TextField, Label e ComboBox

* Criação de tela principal (MainView)

* Criação de lista departamentos (DepartmentList)

* Integração com o JDBC

* Padrão de projetos Observer

* CRUD departamento

* Criação de lista de vendedores (SellerList)

* DatePicker

* CRUD vendedores

* Build de aplicação e preparação para distribuição

* Implantação da aplicação
